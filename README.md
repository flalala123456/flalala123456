<h3 align="left">What I'm working on</h3>

**[Roblox Camera System V4](https://github.com/flalala123456)**
> A modular broadcast-style camera system for Roblox, written in Luau. Includes Steadicam, Spidercam, Dollycam, and Jib modules with velocity-based movement and cam profile-driven tuning.

**[PTCN Camera Whitelist API](https://github.com/flalala123456)**
> A Node.js/Express REST API handling camera access control for Roblox, running on a server hosted by awdev1. Uses SQLite, JobId-based security, and a modular auto-loading route structure. Comes with a totaly professionaly made admin panel.

**[Rust Multiview](https://github.com/flalala123456)**
> A Rust multiview application that reads a color bar along the bottom of the screen each frame to identify which camera is active, then updates the corresponding feed accordingly. Camera switching is handled through OBS Studio.

**[Hardware Camera Switcher](https://github.com/flalala123456)**
> An ATEM-style hardware camera switcher replica built on an Arduino Leonardo — 20 buttons, 10 RGB LEDs, designed in KiCad. Physical control surface for broadcast setups. 3D-printed.
